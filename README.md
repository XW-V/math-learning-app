# 数学启蒙iPad应用

一个专为6岁半儿童设计的数学启蒙Web应用，帮助学习20以内加减法。

## 功能特点

- ✅ **题目生成系统**：随机生成20以内加减法题目，支持自定义难度
- ✅ **数感培养动画**：实物计数动画和数轴动画，帮助理解运算过程
- ✅ **奖励反馈系统**：类似Duolingo的反馈机制，包含音效、动画和连击奖励
- ✅ **计时记分系统**：记录分数、连击、准确率和答题时间
- ✅ **iPad优化**：专为iPad mini优化，触摸友好的界面设计

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 功能说明

### 答题方式
- 使用屏幕上的数字键盘输入答案
- 或直接在输入框中输入数字
- 点击"提交"按钮或按回车键提交答案

### 设置选项
点击右上角的设置按钮可以：
- 选择题目类型（仅加法、仅减法、混合）
- 设置数字范围（1-10 或 1-20）
- 选择动画类型（实物计数、数轴、两者都显示）
- 启用/禁用计时器

### 奖励系统
- **答对**：播放音效、显示星星动画、增加分数
- **连击**：连续答对3题、5题、10题时显示特殊奖励动画
- **答错**：显示正确答案，温和提示

## 注意事项

1. **音效文件**：应用支持音效，但如果没有音频文件，应用仍可正常运行（静默模式）
2. **本地存储**：最佳成绩和设置会保存在浏览器本地存储中
3. **触摸优化**：所有按钮都针对触摸操作进行了优化，最小尺寸44x44px
